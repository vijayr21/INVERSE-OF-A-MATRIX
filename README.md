 Import necessary libraries
### Step 2:

Calculate the inverse of the matrix
### Step 4:

## Program:
```

import numpy as np
A=[[2,1,1],[1,1,1],[1,-1,2]]
inv=np.linalg.inv(A)
print(inv)
```
